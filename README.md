# 🎓 台大補習班 - 智能選位系統

簡單易用的網頁選位系統，學生可快速選位，老師可實時監控和管理。

## ✨ 功能特點

- 👥 **學生選位** - 點擊座位立即確認，一鍵完成
- 🔒 **選位鎖定** - 確認後無法自行更改，避免更換座位
- 📊 **老師後台** - 添加學生、查看選位進度、清除座位
- 👁️ **實時監控** - 黑板排版展示，一目了然誰選了哪個座位
- 💾 **資料匯出** - 支持 CSV、JSON、座位圖片（投影用）
- 📱 **手機友善** - 自適應設計，AB 區並排顯示
- 🔐 **密碼保護** - 老師後台密碼保護，防止誤操作
- ⚡ **實時同步** - 多設備自動同步數據

## 📖 使用說明

### 👨‍🎓 學生端

1. **登入**
   - 輸入姓名
   - 輸入通行碼（例：STU001）
   - 點「開始選位」

2. **選座位**
   - 點擊想要的座位
   - 確認無誤後點「確認選位」
   - ✅ 完成！座位被鎖定，無法更改

### 👨‍🏫 老師後台

密碼：`631015`

#### 📋 學生名單
- 添加新學生：貼入名單，自動分配通行碼（STU001、STU002...）
- 查看學生狀態：已選/待選/已鎖定
- 刪除學生：支持全部刪除或選中刪除

#### 👁️ 選位視圖
- 黑板排版：A 左上、B 右上、C 左下、D 右下
- 實時監控：顯示每個座位的學生姓名
- 點擊座位可清除該學生的選位記錄

#### 💾 資料匯出
- **學生名單** - CSV 格式
- **選位結果** - CSV 格式（包含座位號）
- **完整備份** - JSON 格式（含時間戳）
- **座位圖片** - PNG 投影用（高清 2000×900）

#### ⚙️ 系統設定
- 修改老師密碼
- 輕鬆更改密碼，保護系統安全

## 💡 常見問題

### Q: 學生忘記通行碼怎麼辦？
A: 在老師後台的「學生名單」中查看，每個學生對應的通行碼（STU001、STU002 等）

### Q: 學生選好座位後能改嗎？
A: 不能。確認選位後座位被鎖定，需要聯繫老師在後台清除才能重新選位

### Q: 多設備同時使用會不會有問題？
A: 不會。系統自動同步，多個設備會保持數據一致

### Q: 投影用的座位圖怎麼用？
A: 在「資料匯出」中點「匯出座位圖（投影用）」，下載後可直接投影到黑板
